# Youtube to MP3 Converter

This is a simple web app meant to run locally.
Provide it a link to a youtube video to convert to an mp3.
Build and run instructions will come soon.

## Installation

Install <https://github.com/yt-dlp/yt-dlp> this dependency.

### Windows

1. In a powershell terminal, run the build-windows.ps1 script

### MacOS

1. In a terminal, run the build-macos.sh script

## run

Launch the application. Visit the address printed in the terminal in a browser.
